## OnionHop v1.1.1 - Censored Networks & Resizable UI

### Highlights

- Resizable window and scrollable Settings panel
- Combo box text color fixed in dark mode
- Automatic Exit Location option
- Censored Network Mode (Snowflake/Meek bridges + Secure DNS/DoH)
- Minimize-to-tray triggers on minimize

### Installation

1. Download OnionHop-Setup-1.1.1.exe
2. Run the installer
3. Launch OnionHop and open Settings
4. Enable Auto Update if you want updates to install automatically

### Requirements

- Windows 10/11
- .NET 9.0 Runtime (included in self-contained installer)
