## OnionHop v1.1.4 - WebTunnel + Change Identity

### Highlights

- Add native WebTunnel bridge option (BridgeDB fetch when selected)
- Add "Change Identity" button in Action Center (requests a new Tor circuit)
- Surface WebTunnel note in Settings and About

### Installation

1. Download OnionHop-Setup-1.1.4.exe
2. Run the installer
3. Enable Auto Update if you want update prompts

### Requirements

- Windows 10/11
- .NET 9.0 Runtime (included in self-contained installer)

### SHA-256

- OnionHop-Setup-1.1.4.exe: E074092068B941EB47CA50F7A5152979C21AA1E077F606E84C9BCF2914CD7D23

### Notes

- WebTunnel bridges are fetched from BridgeDB unless you provide custom lines.
- Change Identity requires an active Tor connection and may take a few seconds to reflect.
- Unsigned binaries may trigger antivirus warnings.
- TUN/VPN mode and Kill Switch require Administrator.
